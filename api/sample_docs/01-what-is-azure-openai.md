# What is Azure OpenAI Service

Azure OpenAI Service provides REST API access to OpenAI models hosted in Azure, with the security, compliance, regional availability, and identity integration of the Azure platform. You create a resource in a supported region, deploy one or more models under deployment names you choose, and call them using the OpenAI-compatible REST API or SDKs.

Azure OpenAI supports chat completions (including function calling and streaming), embeddings, image generation, realtime speech, and fine-tuning. Requests are billed per token for language models, with separate rates for prompt (input) and completion (output) tokens.
