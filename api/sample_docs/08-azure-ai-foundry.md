# Azure AI Foundry

Azure AI Foundry is the portal and platform for building, evaluating, and operating AI apps on Azure. It unifies what used to be Azure OpenAI Studio, Azure AI Studio, and prompt flow under a single project-centric workspace. From Foundry you deploy models, build RAG indexes, run evaluations, and manage connections to AI Search and storage.

Foundry also hosts the Agent Service, which is a managed runtime for multi-step agents with tools, threads, and retrieval built in. This quickstart deliberately stays below that layer so the agent loop is legible; upgrading to Agent Service is a clean later step.
